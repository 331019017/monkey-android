# Monkey for GitHub, Android Version
是一个Github第三方客户端，可以查看最新的repo，user，trending。Ios版本的在这里[Monkey](https://github.com/coderyi/Monkey)

## Function
* 登陆 ~~删除~~
* user列表
* repo列表
* trending(daily,weekly,monthly)
* user详情
* repo详情
* follow user
* star repo
* 搜索
